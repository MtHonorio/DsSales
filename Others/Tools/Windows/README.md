# Instalação das ferramentas no Windows

## Ferramentas que você deverá instalar no seu computador:

- Curl
- Git
- Java JDK 11
- Maven
- STS
- Postman
- Postgresql e pgAdmin
- Heroku CLI
- Node & NPM
- VS Code
